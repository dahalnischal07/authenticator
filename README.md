# Secure Authenticator Java Application

A simple **Java-based Authenticator** application with a GUI using **JavaFX**.  
It allows users to input account names and secret keys, and generates One-Time Passwords (OTPs) for secure login.

---

## Features

- Add multiple accounts with secret keys
- Generate placeholder OTPs (for demo)
- Simple, user-friendly GUI using JavaFX
- Easily extendable for TOTP/HOTP integration
- Cross-platform (Windows, Linux, macOS)

---

## Requirements

- Java JDK 17 or higher
- JavaFX SDK
- Git (optional, for version control)
